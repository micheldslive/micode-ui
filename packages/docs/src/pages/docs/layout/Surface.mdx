# Surface

A rounded component for attaching other elements

## Anatomy

Mount all pieces together and create your code

```tsx
import { Surface } from '@micode-ui/react';

<Surface>
  Your code here
</Surface>
```

## Example

<SurfaceDemo />

```tsx
<Surface
  css={{
    display: 'flex',
    alignItems: 'center',
    gap: 20,
    width: '100%',
    height: '60px',
    color: '$black',
    backgroundColor:'$gray700'
  }}
>
  Your code here
</Surface>
```
