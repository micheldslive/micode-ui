# Box

A standard box component to attach other elements

## Anatomy

Mount all pieces together and create your code

```tsx
import { Box } from '@micode-ui/react';

<Box>
  Your code here
</Box>
```

## Example

<BoxDemo />

```tsx
<Box
  css={{
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
    width: '20rem',
    height: '20rem',
    color: '$black',
    backgroundColor:'$gray700'
  }}
>
  Your code here
</Box>
```
