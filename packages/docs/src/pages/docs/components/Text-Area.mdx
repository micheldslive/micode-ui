# Text Area

It is used to collect multi-line user input.

## Anatomy

Mount all pieces together and create your Text Area.

```tsx
import { TextArea } from '@micode-ui/react';

<TextArea />
```

## Example

<HeroContainer>
  <TextAreaDemo />
</HeroContainer>

```tsx
<TextArea placeholder='Leave a short opinion' />
```
