# Tooltip

Display content in a portal hovered a element.

## Anatomy

Mount all pieces together and create your Tooltip

```tsx
import { Tooltip, Box, Text } from '@micode-ui/react';

<Tooltip content="You can now continue shipping">
  <Text>Hover me</Text>
</Tooltip>
```
